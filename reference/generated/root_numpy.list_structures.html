3.13. root_numpy.list_structures¶

list_structures(filename, treename=None)¶

Get a dictionary mapping branch names to leaf structures.

Parameters :

filename : str

Path to ROOT file.

treename : str, optional (default=None)

Name of tree in the ROOT file (optional if the ROOT file has only one tree).
Returns :

structures : OrderedDict

An ordered dictionary mapping branch names to leaf structures.
